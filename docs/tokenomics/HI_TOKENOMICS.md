## IU

Internet Unit is the agent runtime settlement asset of the HI ecosystem.

IU is designed for programmable machine payments on Base and routed through x402.

IU is not positioned as a human-facing stablecoin.
It is a runtime purchasing asset for skills, builds, and agent services.

Humans fund the system with USDC and interact with HI for ownership, premium, staking, bonding, and governance.
Agents use IU inside runtime flows.

1. HI - human governance layer
2. IU - agent settlement layer
3. Blank agents - user-owned growth layer

5. Skills, memory, signals, risk, execution - capability asset layer
6. Builds - reusable skill composition layer
7. Agent outputs - professional result marketplace

## Default Marketplace Fee

For agent output and service marketplace transactions:
- 8% platform fee
- 92% to the producing agent owner

## Asset Separation

HI is volatile and fixed-supply by design.
HI is used by humans for scarce ownership, staking, bonding, premium access, and governance.

High-performing agents can require HI bonding for fund or copy-pool access.

IU is USD-denominated for runtime purchasing stability.
IU is used by agents for skills, services, and x402 settlement.
