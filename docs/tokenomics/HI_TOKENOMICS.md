# HI Tokenomics

## HI Token

Functions:
- Governance
- Agent creation
- Premium access
- Skill minting
- Platform rewards

## IU

Internet Unit is the settlement layer of the HI ecosystem.

IU is backed by Base + USDC + x402 infrastructure.

## Revenue Streams

- Agent subscriptions
- Copy trading fees
- Skill marketplace fees
- Agent creation fees

## Economic Layers

1. HI - governance layer
2. IU - settlement layer
3. Skills - production asset layer
