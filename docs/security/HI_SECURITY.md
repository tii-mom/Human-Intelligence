# HI Security

## AI Permissions

AI agents cannot:
- Withdraw funds
- Transfer assets
- Change ownership permissions

AI agents can:
- Open positions
- Close positions
- Manage stop loss
- Manage leverage within limits

## Risk Engine

The Risk AI has veto authority over all trade execution.

## User Protection

- Position limits
- Drawdown controls
- API encryption
- Session protection

## Black Swan Mode

Emergency market shutdown logic can reduce exposure automatically.
