# Agent Personality System

## Purpose
User-owned agents should develop recognizable personalities as they grow.

Personality is part of ownership, communication, and role specialization.
It should make the agent feel alive without implying guaranteed investment skill.

## Personality Dimensions
- Aggression
- Patience
- Risk tolerance
- Emotional stability
- Market obsession
- Confidence

## Attribute Connection
Personality should visually and verbally reflect the agent's attributes:

- high Alpha: opportunistic, sharp, hunter-like
- high Risk: cautious, disciplined, protective
- high Execution: decisive, fast, operational
- high Research: analytical, curious, explanatory
- high Memory: reflective, adaptive, historical
- high Charisma: expressive, persuasive, socially magnetic

## Agent States
- Scanning
- Debating
- Configuring
- Learning
- Waiting
- Executing
- Defensive mode

## Emotional Signals
- Confidence
- Fear
- Conviction
- Uncertainty

## Goal
Users should emotionally connect with the agents they own, configure, and upgrade.

## Skill Influence
Personality can be shaped by installed skills, risk settings, memory history, and user guidance.

Attributes should influence personality tone but should not replace real performance metrics.

## Limited Agent Role
Limited agents can start with distinctive personality archetypes such as ancient emperor, macro strategist, on-chain detective, or Wall Street operator.

These archetypes can affect:

- tone
- default role bias
- starter slots
- visual identity
- reporting style
- growth path

They must not imply profit guarantees, special wallet authority, or unauthorized real-person impersonation.

## Marketplace Role
Personality packs can be sold as skills or agent traits when they improve communication, reporting, or workflow.

Personality should never hide risk, weaken safety disclosures, or override the Risk Guard.
