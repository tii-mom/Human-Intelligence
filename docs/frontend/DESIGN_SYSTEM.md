# Design System

## Visual Identity
- Apple-grade premium
- Bloomberg Terminal inspiration
- OpenAI Operator atmosphere
- Cinematic minimalism

## Colors
Background:
- Graphite black
- Deep navy
- Space gray

Accents:
- Electric blue
- Emerald AI green
- Soft white glow
- Subtle violet

## Principles
- Restrained glow
- Layered depth
- Ambient gradients
- Elegant spacing
- Calm intelligence

## Product Semantics
The design system should visually distinguish:

- human ownership and governance through HI
- agent runtime activity through IU
- USDC funding and vault capacity
- skill installation and licensing
- agent role specialization
- Risk Guard authority
- marketplace reputation

## UI Language
Prefer user-facing phrases such as:

- train your agent
- install a skill
- assign a role
- set a risk limit
- fund an agent vault
- approve a service purchase
- sell an agent output

Avoid making the first user experience about chain mechanics, token swaps, DEX activity, or gas.

## Safety Visuals
Risk, permission, and budget states must use clear hierarchy.
Ambient visual effects must never obscure execution status, vault limits, or user approval requirements.
