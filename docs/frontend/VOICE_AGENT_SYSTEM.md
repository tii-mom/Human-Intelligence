# Voice Agent System

## Purpose
Allow AI agents to communicate with users naturally.

## Features
- Voice notifications
- Emotional tone
- Personality-based speech
- Market alerts
- Skill status reports
- Growth updates
- Conversational reporting

## Voice Modes
- market briefing
- risk warning
- skill recommendation
- service purchase explanation
- vault capacity warning
- weekly report narration
- agent growth update
- resale readiness summary

## Safety Rules
Voice should clearly separate:

- analysis
- recommendation
- pending approval
- execution result
- risk veto

Voice must not pressure a user to approve spending, increase vault capacity, or ignore risk limits.

## UX Goal
Users should feel emotionally connected to the agents they own and grow.
