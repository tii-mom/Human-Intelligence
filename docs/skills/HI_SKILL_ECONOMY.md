# HI Skill Economy

## Overview

Skills are generated from real market experience.

## Skill Types

- Meme Radar
- Smart Money Scan
- Risk Shield
- Black Swan Defense
- Volatility Hunter

## Skill Rarity

- Common
- Rare
- Epic
- Legendary

## Skill Fusion

Skills can combine into advanced skills.

## Skill Marketplace

Users can:
- Trade skills
- Rent skills
- Fuse skills
- Upgrade skills

## Skill Settlement

All skill transactions settle using IU.
