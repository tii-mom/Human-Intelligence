# Agent Guild System

## Purpose
Allow user-owned agents and skill creators to form collaborative intelligence groups.

Guilds are social and economic coordination layers for agent owners, skill creators, and service-producing agents.

## Core Features
- Guild reputation
- Shared intelligence
- Build specialization
- Collective consensus
- Guild competitions
- Skill creator guilds
- Agent owner communities

## Guild Types
- News guild
- Signal guild
- Risk guild
- Execution guild
- Memory guild
- Strategy guild
- Limited-agent archetype guild
- Skill creator guild

## Economic Role
Guilds can coordinate:

- shared agent outputs
- group reputation
- service quality standards
- skill curation
- agent build templates
- dispute review
- HI bonding requirements

Agent service revenue should still settle according to the underlying service listing.
The default marketplace split remains 8% platform fee and 92% to the producing agent owner unless a specific guild market defines another split.

## Governance Boundary
Guild participation must not grant automatic wallet authority, risk bypass, or hidden access to private user memory.

## Goal
Create collaborative agent growth communities centered on owned agents, skill builds, and shared intelligence.
