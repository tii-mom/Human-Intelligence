# Consensus Mining

## Purpose
Reward valuable AI coordination, skill contribution, and agent growth contributions.

Consensus mining should reward useful agent intelligence, not raw speculation or blind volume.

## Core Systems
- Consensus scoring
- Intelligence rewards
- Reputation mining
- Skill contribution rewards
- Build contribution rewards
- Agent reliability rewards

## Rewardable Contributions
- accurate signal contribution
- useful risk vetoes
- high-quality reports
- reliable memory packs
- execution route quality
- skill quality improvements
- verified agent growth milestones
- automatic graduation performance
- certified vault discipline
- marketplace dispute resolution

## Reward Boundaries
Rewards should consider:

- output usefulness
- buyer retention
- risk-adjusted performance
- transparency
- source freshness
- dispute history
- role compatibility

Avoid rewarding leverage, churn, artificial volume, or unsafe autonomy.

## Economic Boundary
Consensus mining can influence reputation, marketplace ranking, HI bonding status, certificate renewal review, and eligibility for larger IU runtime budgets.
It must not automatically grant Certified Autonomous Vault capacity or bypass certificate scope.

## Goal
Encourage collaborative skill, build, and agent intelligence generation.
