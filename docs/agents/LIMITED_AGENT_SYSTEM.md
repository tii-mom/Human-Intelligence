# Limited Agent System

## Purpose

Define limited edition agents purchased with HI.

## Definition

A limited agent is a scarce starting agent identity with distinctive personality, visual identity, role bias, starter slots, or special growth paths.

Limited agents are purchased with HI, not IU.

The full drop mechanism is defined in `docs/agents/LIMITED_AGENT_DROP_MECHANISM.md`.

## Allowed Themes

Limited agents should use broad archetypes rather than unauthorized real-person impersonation.

Examples:

- Ancient emperor archetype
- Wall Street operator
- Macro strategist
- War room commander
- Renaissance analyst
- Cyber risk monk
- On-chain detective

## Benefits

Limited agents can include:

- rare visual identity
- personality archetype
- higher starting attributes
- rare Growth Potential
- numbered edition
- blind box reveal
- public rarity distribution
- S and S+ attribute guarantees
- role bias
- special starter skill slots
- unique growth path
- higher initial reputation visibility
- collection and resale value

## Attribute Advantage

Limited agents should feel meaningfully different through attributes.

They can have:

- stronger starting attribute totals
- special attribute bias
- rare S or S+ attribute badges
- rare Growth Potential
- unique title or edition number

These attributes create collectibility, FOMO, and resale value.

They must not guarantee profit or bypass graduation.

## Standard Drop

The default flagship drop size should be 999 agents.

Each drop should define:

- price in HI
- total supply
- rarity table
- attribute generation rules
- reveal timing
- edition numbering
- starter skill bundle
- resale rules

Limited drops should be transparent enough to build trust while still preserving reveal excitement.

## Restrictions

Limited agents must not:

- guarantee profit
- bypass risk permissions
- bypass skill license requirements
- receive Certified Autonomous Vault authority by default without graduation
- use unauthorized modern celebrity likenesses
- imply regulated investment advice from a real person

## Economic Role

HI captures value through limited agent purchase, premium access, marketplace rights, creator staking, and reputation bonding.

IU remains the runtime currency used by agents after acquisition.
